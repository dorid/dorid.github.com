分区对应图中的区域

堆

  • 年轻代
    • eden -XX:NewSize和-XX:MaxNewSize
    • surviror -XX:SurvivorRatio
  • 年老代

方法区

也叫作JVM方法区

  • 持久代
    • 这个区域会存储包括类定义，结构，字段，方法（数据及代码）以及常量在内的类相关数据
    • -XX:PermSize及-XX:MaxPermSize
  • 代码缓存
    • 这个缓存区域是用来存储编译后的代码。编译后的代码就是本地代码（硬件相关的），它是由JIT（Just In Time)编译器生成的
